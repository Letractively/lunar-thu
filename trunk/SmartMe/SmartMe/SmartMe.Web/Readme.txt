=========================
SmartMe.Web is integration for web resources on the basis of :
1. use search engine, and human knowledge database for useful informations
2. crawl web for resources.
3. filter the informations

It should be:
1. easy to start a query, and easy to terminate.
2. tolerant to network error.
3. avoid taking up too much CPU, memory resources.
